# **01 프로젝트 목표 및 배경**

<aside>
📌

**프로젝트 목표**

- **성과 검증 지원**: 가우시움 청소로봇 데이터를 기반으로 매월 **사용량 리포트**를 발행, 고객사 관리자가 **도입 성과를 수치로 검증**하고 **지속·확장 도입 여부**를 객관적으로 판단할 수 있도록 지원
- **운영 최적화 기여**: 관리자가 로봇의 **운영 효율성·자원 소모 패턴·비용 절감 효과**를 확인하여, **현장 운영 및 유지보수 계획**에 활용 가능
- **자사 경쟁력 강화**: 솔링크를 단순 모니터링 도구가 아닌 **성과 검증·의사결정 지원 플랫폼**으로 포지셔닝, 경쟁사 대비 차별화 확보
- **매출 확대 기반**: 객관적 수치와 축적된 데이터를 토대로 기존 고객사에는 **추가 도입을 설득**하고, 업종·현장별 **벤치마크 제공**을 통해 신규 고객 확보까지 연결
</aside>

# **02 용어 규정**

| **용어** | **정의** |
| --- | --- |
| 운영 로봇 현황 | 해당 월의 로봇 운영 현황을 가동 여부에 따라 구분하여 제공하는 콘텐츠 |
| 로봇 활용도 | 해당 월에 로봇이 얼마나 활용되었는지 파악할 수 있는 데이터를 제공하는 콘텐츠 |
| 로봇 운영 성과 | 해당 월에 로봇이 얼마나 효과적으로 활용되었는지 파악할 수 있는 데이터를 제공하는 콘텐츠 |
| 운영 안정성 | 해당 월에 로봇이 얼마나 안정적으로 작동하였는지 파악할 수 있는 데이터를 제공하는 콘텐츠 |
| 운영 자원 소모 | 해당 월에 로봇이 각종 운영 자원 (배터리, 정수, 소모품)을 얼마나 사용하였는지 파악할 수 있는 데이터를 제공하는 콘텐츠 |

# **03 요구사항**

## **정책**

| **구분** | **상세** |
| --- | --- |
| **월간 리포트 집계∙발행 단위** | - 월간 리포트는 **`하나의 사이트`**를  대상으로 데이터를 집계하여 발행한다. |
| **리포트 발송 일자** | - 리포트는 한국표준시(KST)를 기준으로, 매월 1일 오전 9시에 발송한다. |
| **집계 기간**  | - 월간 리포트용 데이터 집계 기간은 모든 사이트에 동일하게 적용한다.
- 리포트 발송 일자를 기준으로 다음 기간 동안의 가우시움 청소 로봇 사용량을 집계한다:
`전월 1일 00:00:00 (KST) - 전월 말일 23:59:59 (KST)` |
| **증감 정보 제공** | - 증감 정보로는, 증감량 또는 증감률을 제공한다.
- 증감량은 다음과 같이 집계한다. 데이터 표기 정책은 ‘데이터 표기’ 항목을 확인한다.
`$증감량 = {(해당 월 수치 - 전월 수치)}$`
- 증감률은 다음과 같이 집계한다. 데이터 표기 정책은 ‘데이터 표기’ 항목을 확인한다.
**`$증감률 = \frac{(이번 달 수치 - 전월 수치)}  {(전월 수치)} \times 100$`** |
| 비가동 로봇  | - 집계 해당 월 기준, 다음 항목에 대해 단 한 건의 이력도 없는 로봇은 비가동 로봇으로 정의한다.
- 고객이 어떤 로봇이 가동하지 않았는지 알 수 있도록, 비가동 로봇의 정보 (로봇명, 디바이스 ID)를 명시한다. |
| 집계 대상 제외 처리 | - 가우시움 청소로봇이 한 대도 등록되지 않은 사이트는, 리포트 발행 대상에저 제외한다.
- 집계 해당 월 기준 3개월간 다음 항목에 대해 `단 한 건의 이력도 없는 로봇`은, 월간 리포트용 데이터를 집계하지 않는다.
1) 청소 작업
2) Incident 발생 로그

- 비가동 로봇은 다음과 같이 그 정보를 별도로 명시한다.
1) 로봇명, 디바이스 id |
| **데이터 표기** | - 데이터 표기 정책은 다음과 같다.
1) 모든 데이터는 공통적으로 소숫점 1의 자리에서 반올림하여, 정수 형태로 표기한다. 단, 증감량과 증감률은 별도의 정책에 따라 표기한다.
2) 청소 면적은 제곱미터 ($m^2$) 단위로 표기한다.
3) 시간은 X 시간 Y 분 Z 초로 표현한다. 단, 0의 단위가 나올 경우 표시하지 않는다.
예시) 260938 초 → 72 시간 28분 58초
         3620 초 → 1시간 20초
4) 비율은 백분율 (%) 형식으로 표현한다.
5) 최종 표기값이 0이어도 그대로 표기한다.

- 증감량과 증감률 표기 정책은 다음에 따른다.
1) 해당 월 기준 전월 데이터가 0이거나 없는 경우, 증감량과 증감률을 모두 ‘-’ 로 표기한다.
2) 리포트 해당 월과 전월 데이터가 동일한 경우, 증감량과 증감률을 모두 ‘-로 표기한다.
3) 증감량 또는 증감률이 10 이상일 경우 소숫점 첫번째 자리에서 반올림하여 `정수` 형태로 표기한다.
4) 증감량 또는 증감률이 1 이상일 경우 소숫점 두번째 자리에서 반올림하여 소숫점 첫번째 자리까지 표기한다.
5) 증감량 또는 증감률이 1 미만일 경우 소숫점 세번째 자리에서 반올림하여 소숫점 두번째 자리까지 표기한다. |

## **핵심 요구사항**

- **임의의 사이트를 대상으로, 다음의 네가지 항목으로 구성된 청소로봇 운영 리포트를 작성한다.**
    - **0. 운영 로봇 현황**
        
        해당 월에 사이트에서 운영된 로봇의 현황을 다음과 같이 구분하여 제공한다
        
        - 운영 로봇 대수 합계
        - 가동 로봇
            - 가동 로봇 대수 합계
        - 비가동 로봇
            - 비가동 로봇 대수 합계
            - 비가동 로봇별 로봇명과 디바이스 ID
    - **1. 로봇 활용도**
        
        청소 면적, 작업 시간, 작업 건수 세가지 지표별로 각각 합계를 집계하여 제공한다. 이때 각 지표별 추이를 집계하여 함께 제공한다.
        
        - **청소 면적**
            - 실 청소 면적을 합산하여 합계를 표기한다. 이때, 전월 대비 증감량과 증감률을 집계하여 함께 표기한다.
            - 집계 대상 월 (전월)을 포함하여 지난 3개월간의 청소 면적을 각각 합산하여, 그 추이를 막대 그래프와 꺾은선 그래프를 결합한 형식로 표기한다.
        - **작업 시간**
            - 작업 시간을 합산하여 합계를 표기한다. 이때, 전월 대비 증감량과 증감률을 집계하여 함께 표기한다.
            - 집계 대상 월 (전월)을 포함하여 지난 3개월간의 작업 시간을 각각 합산하여, 그 추이를 막대 그래프와 꺾은선 그래프를 결합한 형식로 표기한다.
        - **작업 건수**
            - 청소 작업 건수를 합산하여 합계를 표기한다. 이때, 전월 대비 증감량과 증감률을 집계하여 함께 표기한다.
            - 집계 대상 월 (전월)을 포함하여 지난 3개월간의 작업 건수를 각각 합산하여, 그 추이를 막대 그래프와 꺾은선 그래프 형태로 표기한다.
    - **2. 로봇 운영 성과**
        
        총 작업 성공 건수, 작업 성공률, 청소 완수율, 작업당 청소 완수율, 청소 효율을 각각 집계하여 제공한다. 
        이때 각 지표별 추이를 집계하여 함께 제공한다.
        
        - **작업 성공 건수**
            - 완수율 (completionPercentage) 기준 0.8 이상인 작업을 합산하여 합계를 표기한다. 이때, 전월 대비 증감량과 증감률을 집계하여 함께 표기한다.
            - 해당 월을 포함하여 지난 3개월간의 작업 성공 건수를 각각 합산하여, 그 추이를 막대 그래프와 꺾은선 그래프를 결합한 형식로 표기한다.
        - **작업 성공률**
            - 작업 건수 합계 대비 작업 성공 건수 합계가 차지하는 비율을 계산하여 백분율 형식으로 표기한다.이때 전월 대비 증감량과 증감률을 집계하여 함께 표기한다.
            - 해당 월을 포함하여 지난 3개월간의 작업 성공률을 각각 계산하고, 그 추이를 막대 그래프와 꺾은선 그래프를 결합한 형식로 표기한다.
        - **청소 완수율**
            - 계획 청소 면적 합계 대비 실 청소 면적 합계가 차지하는 비율을 계산하여 백분율 형식으로 표기한다. 이때 전월 대비 증감량과 증감률을 집계하여 함께 표기한다.
            - 해당 월을 포함하여 지난 3개월간의 청소 완수율을 각각 계산하고, 그 추이를 막대 그래프와 꺾은선 그래프를 결합한 형식으로 표기한다.
        - **작업당 청소 완수율**
            - 작업별 청소 완수율을 작업 건수 합계로 나누어 값을 표기한다. 이때 전월 대비 증감량과 증감률을 집계하여 함께 표기한다.
            - 해당 월을 포함하여 지난 3개월간의 작업당 청소 완수율을 각각 계산하고, 그 추이를 막대 그래프와 꺾은선 그래프를 결합한 형식으로 표기한다.
        - **청소 효율**
            - 실 청소 면적 합계를 작업 시간 합계로 나누어 값을 표기한다. 단위는 시간으로 표기한다. 이때 전월 대비 증감량과 증감률을 집계하여 함께 표기한다.
            - 해당 월을 포함하여 지난 3개월간의 청소 효율을 각각 계산하고, 그 추이를 막대 그래프와 꺾은선 그래프를 결합한 형식으로 표기한다.
    - **3. 운영 안정성**
        
        총 오류 발생 건수, 작업당 오류 발생 건수, 오류 유형별 분포를 각각 집계하여 제공한다. 이때 각 지표별 추이를 집계하여 함께 제공한다.
        
        - **총 오류 발생 건수**
            - 오류 발생 건수를 합산하여 합계를 표기한다. 이때, 전월 대비 증감량과 증감률을 집계하여 함께 표기한다.
            - 또한, 총 오류 발생 건수를 다음의 항목으로 분류하여 표기한다. 이때 각 항목은 전월 대비 증감량과 증감률을 집계하여 함께 표기한다.
                - 성공 작업 기준 오류 발생 건수
                - 실패 작업 기준 오류 발생 건수
                - 작업 외 오류 발생 건수
            - 해당 월을 포함하여 지난 3개월간 다음의 항목을 각각 계산하고, 그 추이를 막대 그래프와 꺾은선 그래프를 결합한 형식으로 표기한다.
                - 성공 작업 기준 오류 발생 건수
                - 실패 작업 기준 오류 발생 건수
                - 작업 외 오류 발생 건수
        - **작업당 오류 발생 건수**
            - 작업 중 오류 발생 건수 합계를 작업 건수 합계로 나누어 값을 표기한다. 이때, 전월 대비 증감량과 증감률을 집계하여 함께 표기한다.
            - 또한, 최소값, 최대값, 중위값을 제공하여 작업 중 오류 발생 건수 분포를 파악할 수 있도록 한다.
            - 해당 월을 포함하여 지난 3개월간의 작업당 오류 발생 건수를 각각 계산하고, 그 추이를 막대 그래프와 꺾은선 그래프를 결합한 형식으로 표기한다.
        - **오류 유형별 분포**
            - 당월 기준 오류 분포 집계
                - 해당 월의 오류 발생 건수 집계하여, 발생 건수가 가장 많은 상위 5개의 오류 유형을 식별한다.
                - 각 오류 유형별로 다음 항목을 집계하여 제공한다.
                    1. 발생 건수 합계
                    2. 실패 작업 기준 발생 건수 합계
                    3. 오류 비중
                        
                        전체 오류 발생 건수 합계 대비 해당 오류 발생 건수 합계
                        
            - 3개월 기준 오류 분포 집계
                - 해당 월을 포함하여 지난 3개월간의 오류 발생 건수를 합산하여, 발생 건수가 가장 많은 상위 5개 오류 유형을 식별한다.
                - 각 오류 유형별로 다음 항목을 집계하여 제공한다.
                    1. 발생 건수 합계
                    2. 실패 작업 기준 발생 건수 합계
                    3. 오류 비중
                        
                        전체 오류 발생 건수 합계 대비 해당 오류 발생 건수 합계
                        
    - **4. 운영 자원 소모**
        
        배터리 사용량, 정수 사용량, 소모품 사용량 각 항목별로 합계와 단위당 사용량을 집계하여 제공한다. 이때 각 지표별 추이를 집계하여 함께 제공한다.
        
        - 배터리 소모율
            - 작업당 배터리 소모율
                - 배터리 소모율 합계를 작업 건수 합계로 나누어 값을 표기한다. 이때, 전월 대비 증감량과 증감률을 집계하여 함께 표기한다.
                - 또한, 최소값, 최대값, 중위값을 제공하여 배터리 소모율 분포를 파악할 수 있도록 한다.
