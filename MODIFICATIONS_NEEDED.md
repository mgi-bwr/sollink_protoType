# 소모품 관리 프로토타입 수정사항

## 요청된 수정사항

1. 모달 → 상세 페이지 전환
   - `openDetailModal` → `openDetailPage`로 함수명 변경
   - 모달 구조 제거, 새로운 상세 페이지 레이아웃 추가

2. 각 소모품별 저장 버튼 추가
   - 각 소모품 카드에 개별 저장 버튼 추가
   - `saveConsumable(typeId, typeName)` 함수 구현

3. 미설정 소모품 처리
   - 모든 소모품의 입력창 항상 표시
   - 설정되지 않은 소모품에 "소모품 정보가 설정되지 않았습니다" 경고 표시

4. 초기화 버튼 배치
   - 입력창과 같은 행에 배치
   - 4열 그리드 구조로 변경 (3개 입력 + 1개 버튼)

5. 이력 조회 버튼 배치
   - 입력창 바로 아래 위치
   - 저장 버튼과 같은 행에 배치

## 구현 방법

파일이 크기 때문에 다음 단계로 수정이 필요합니다:
1. HTML 구조 변경
2. JavaScript 함수 수정
3. CSS 스타일 조정
